#  Personal Portfolio Website

A modern and responsive portfolio website showcasing my skills, projects, and experience as a **Data Scientist & AI Enthusiast**.

---

## 🚀 About the Project

This portfolio is designed to present my work, technical skills, and achievements in a clean and interactive way. It features smooth animations, dark/light theme switching, and a visually appealing UI.

---
##  Features

* 🌙 Dark & Light Mode Toggle
* ✨ Smooth Animations & Transitions
* 👤 Hero Section with Profile Image
* 📊 Skills Visualization
* 💼 Projects Showcase
* 📜 Resume View & Download
* 🏆 Certificates Section with Modal Preview
* 📱 Fully Responsive Design

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Custom CSS (Glassmorphism + Animations)
* **Tools:** Git, GitHub
